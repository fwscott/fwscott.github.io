FAQ for tour guides

What if I run two tours back-to-back?

VoyaReach uses one active roster at a time. Start your next trip and the prior group is automatically archived so messages route to the correct current guests.

Do guests need to download an app?

No. Guests use their existing WhatsApp. Only tour guides and operators log into the VoyaReach dashboard.
